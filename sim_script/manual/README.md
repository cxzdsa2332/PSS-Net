# Manual and exploratory scripts

Scripts in this directory are historical prototypes, one-off checks, or
exploratory plotting scripts. They are useful for tracing method development,
but they are not formal reproduction entry points.

When a script becomes part of the manuscript workflow, move or rewrite it under
one of the formal `sim_script/0*_*/` folders for numeric simulation output and
put the corresponding plotting code in `analysis_script/`.
